## Beaten beat v1.0.0
함께 플레이 리스트를 만들고, 듣고 싶은 노래에 투표하세요!

- 공용 공간에서 노래를 틀고 싶을 때, 사용할 수 있는 서비스입니다.

- 유저의 투표 기능을 통해 플레이 리스트 순서가 실시간으로 변경됩니다.
